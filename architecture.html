1 The AI Engine Lives Inside app.js

The AI is a JavaScript object called AI defined at line 87 of app.js. It contains 12+ intelligent functions, each designed for a specific purpose. This is not a separate server or external service — it runs directly in the user's browser.

3 AI Reads User Data from State (S)

The AI engine is deeply connected to the State object (S). It reads the user's tasks, moods, expenses, cycle data, energy level, sleep logs, and more to make personalized decisions:

// AI.getDashInsight() reads EVERYTHING to generate one smart sentence: getDashInsight() { const pending = S.tasks.filter(t => !t.done).length; // how many tasks? const todayMood = S.moods.find(m => m.date === today); // how is she feeling? const total = S.expenses.reduce(...); // spending status? const phase = this.getCyclePhase(); // cycle phase? // AI makes decisions based on ALL this data combined: if (todayMood === 'stressed') → "Adjusted suggestions for more breaks" if (phase === 'menstrual') → "Schedule lighter tasks, prioritize rest" if (spending > 70%) → "⚠️ Review your expenses" }

This means every response is personalized to the specific user. Two different women using the app will get completely different AI advice based on their unique data.

6 External LLM Integration (The Upgrade Path)

When a user adds an API key in Settings, the AI Chat automatically upgrades from built-in to external LLM. The system prompt feeds all user data into the LLM:

// The system prompt makes the LLM "become" HER-OS (line 1225) const systemPrompt = `You are HER-OS, an AI Life Operating System for women. You help with productivity, scheduling, finance, wellness, learning, home management, leadership, and personal branding. The user's data: Energy: ${S.user.energy} Pending tasks: ${S.tasks.filter(t => !t.done).length} Budget remaining: ₹${remaining} `; // Sent to Groq or OpenAI API fetch(url, { body: JSON.stringify({ model: 'llama-3.3-70b-versatile', messages: [ { role: 'system', content: systemPrompt }, { role: 'user', content: userMessage } ] }) });

Fallback safety: If the API call fails, the system automatically falls back to the built-in AI engine, so the user is never left without a response.

// Graceful fallback (line 1242-1244) catch (e) { const fallback = AI.chat(msg); // built-in AI takes over }
