How We Built HER-AI

An entire Life Operating System created with just 3 AI prompts. No months of coding. No complex tools. Here's exactly how we did it — and how you can run it on your laptop in under 2 minutes.

Development Process

Built with 3 Prompts. That's It.

We used Cursor IDE (an AI-powered code editor) to build the entire system through just 3 strategic prompts — proving that powerful apps can be created rapidly by anyone.

1
Prompt 1 — The Foundation
Design the full Life Operating System
"Build a complete Life Operating System for women called HER-AI with 8 modules: Productivity (energy-based task management), Smart Scheduler (cycle-aware planning), Learning (AI instructor with courses), Home Life (grocery, meals, geo-reminders), Finance (budgeting & tracking), Leadership (email rewriting, negotiation), Branding (LinkedIn, pitches), and Wellness (mood, sleep, cycle tracking). Use vanilla HTML/CSS/JS, no frameworks. Include a landing page and a dashboard app. Use Groq LLM with rule-based fallback."
Output: Complete app with all 8 modules, landing page with animations, dashboard with AI insights, hybrid AI engine, full CSS design system with light/dark themes.
16 JS modules Landing Page Dashboard AI Engine 4 Courses Cycle Tracking
2
Prompt 2 — Intelligence Layer
Make the AI contextual and personalized
"Connect the AI chat to live user data — tasks, budget, moods, energy level, cycle phase, schedule. The AI should give specific answers based on actual data, not generic advice. Add energy-based task highlighting (recommended vs defer). Fix geo-reminders to use real browser geolocation with OpenStreetMap reverse geocoding instead of random simulation."
Output: Personalized AI responses using real data, energy-adaptive task UI with visual indicators, real GPS-based geo-reminders matching store categories.
Live Data Context Energy UI Real Geolocation Smart Fallbacks
3
Prompt 3 — Polish & Deploy
Make it production-ready and shareable
"Set up the project for deployment — add GitHub repo, push all code, write a comprehensive README, configure auto-launch with VS Code tasks so pressing F5 starts the server and opens the browser automatically. Add architecture documentation and a presentation page."
Output: GitHub-hosted repo, one-click launch, full README with setup instructions, architecture docs, presentation page.

Follow These Steps

Step 1 Download the App

Go to the project page and click the green "Code" button, then click "Download ZIP":

github.com/Gayatri-Kharat/HER-AI

Once downloaded, right-click the ZIP file and choose "Extract All". Open the extracted folder.

It's just like downloading and unzipping any file from the internet!

↓
Step 2 Install Python (One-Time Only)

If you don't have Python yet, download it for free:

python.org/downloads

Click "Download Python" and run the installer. Important: Check the box that says "Add Python to PATH" before clicking Install.

You only need to do this once. After that, Python is always ready on your laptop.

↓
Step 3 Open the App

On Windows: Open the HER-AI folder, click the address bar at the top, type cmd, and press Enter. This opens a black window.

On Mac: Right-click the HER-AI folder and choose "Open Terminal here".

Then type this and press Enter:

python -m http.server 5500

You'll see a message saying "Serving HTTP..." — that means it's working! Keep this window open.

↓
Step 4 Start Using HER-AI!

Open your browser (Chrome, Edge, etc.) and type this in the address bar:

localhost:5500

You'll see the HER-AI landing page! Click "Launch App" to open your personal dashboard.

Bookmark this page so you can find it easily next time.

↓
Optional Turn on Smart AI Chat

Want the AI chatbot to give personalized, human-like responses? Here's how:

1. Go to console.groq.com and sign up for free

2. Click "API Keys" and create a new key (it's a long text code)

3. In the HER-AI app, go to Settings (gear icon in sidebar) and paste the key there

Without this, the app still works perfectly — it just uses simpler built-in responses instead of the AI.
