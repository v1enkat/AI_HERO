# Archive

**`legacy/`** — Original single-page vanilla HTML/JS/CSS version of the app. It is **not** used by the React + Node stack (`client/` and `server/`). Keep it only if you want a reference or might open `archive/legacy/app.html` locally.

To run the old demo from disk, use a local server in `archive/legacy/` and copy `config.example.js` to `config.js` there if the UI expects a Groq key.
